Task Management System Functionalities


1. use valid account to login & logout system

2. click 'New Job' button to create new jobs, default job status is 'unassigned', new job will be update into DB

3. drag & drop job to assign job to different user or change start time, all changes will be update into DB

4. drag a scheduled job to re-schedule area for unsigned

5. click 'Day' 'Week' 'Month' button to switch different pages

6. select different job status to filter out showing result, it allows multiple selection
